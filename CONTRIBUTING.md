# How to contribute

I'm really glad you're reading this, because we need volunteer developers to help this project come to fruition.

Here are some important resources:

  * [discord](https://discord.gg/jbfQZfM) tells you where we are,
Пожалуйста, обратитесь к руководству по стилю и вкладу каждого проекта для отправки исправлений и дополнений. В общем, мы следуем рабочему процессу Git "вилки и вытягивания".

Форк репо на GitHub
Клонируйте проект на свой компьютер
Зафиксируйте изменения в своей ветке
Верните свою работу обратно на вилку
Отправьте запрос на извлечение, чтобы мы могли рассмотреть ваши изменения
ПРИМЕЧАНИЕ. Не забудьте объединить последние данные из «восходящего потока» перед выполнением запроса на перенос!
